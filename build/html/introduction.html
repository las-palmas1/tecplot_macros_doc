Introduction¶

Модуль tecplot_lib является оболочкой для некоторых функций и команд скриптового языка TecPlot. Он преимущественно представляет собой совокупность функций, каждая из которых возвращает форматированную строку, содержащую определенную команду или их набор на языке TecPlot. Возвращаемые ими строки складываются и записываются в текстовый файл с расширением .mcr, который запускается командой из командной строки. С помощью функций и классов, содержащихся в нем, можно сделать следующее:

  1. Извлечь данные из файла данных TecPlot по произвольной полилинии и сохранить их в текстовом файле
  2. Считать данные из текстового файла в переменную типа pandas.DataFrame
  3. Создать новые переменные в файле данных TecPlot, используя синтаксис уравнений TecPlot
  4. Создать изображение произвольно расположенного среза, с регулированием настроек легенды, шрифтов, распределения цвета, переменной и т.д.

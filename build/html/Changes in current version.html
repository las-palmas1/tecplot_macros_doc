Changes in current version¶

В текущей версии библиотеки (версия 0.4) по сравнению с версией 0.3 был изменен порядок параметров в инициализаторе класса PictureCreator. В силу этого код, написанный для предыдущей версии, может оказаться невалидным для текущей версии.

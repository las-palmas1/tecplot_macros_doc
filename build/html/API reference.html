Параметры:
  • x_axis_var –

    int

    номер переменной, откладываемая по горизонтальной оси, например, x_axis_var = 0

  • y_axis_var –

    int

    номер переменной, откладываемая по вертикальной оси

  • rect –

    tuple, optional

    определяет положение прямоугольника сетки на frame, rect=(x1, y1, x2, y2), по умолчанию rect=(10, 10, 90, 90)

    bool, optional

    Сохраняемость масштаба осей при изменении их диапазона

  • x_line_pos –

    float

    Позиция горизонтальной оси по вертикали

  • y_line_pos –

    float

    Позиция вертикальной оси по горизонтали

  • x_title_font –

    Font, optional

    экземпляр класса Font, содержащий настройки шрифта для заголовка оси x

  • x_label_font –

    Font, optional

    экземпляр класса Font, содержащий настройки шрифта для лэйблов оси x

  • x_title_offset –

    float, optional

    сдвиг заголовка оси x относсительно оси

  • x_label_offset –

    float, optional

    оступ подписей оси x от оси

  • y_title_font –

    Font, optional

    экземпляр класса Font, содержащий настройки шрифта для заголовка оси y

  • y_label_font –

    Font, optional

    экземпляр класса Font, содержащий настройки шрифта для лэйблов оси y

  • y_title_offset –

    int, optional

    сдвиг заголовка оси y относсительно оси

  • y_label_offset –

    float, optional

    оступ подписей оси y от оси

  • kwargs – xlim и ylim (интервалы по осям x и y соотвественно), тип tuple; пример: xlim=(0,1), ylim=(1,2)

class tecplot_lib.LineDataExtractor(datafiles_dir, output_dir, polylines_list: typing.List[typing.List[tecplot_lib.PolyLine]], macro_name)[исходный код]¶
Базовые классы: object

Обеспечивает возможность создания макроса, извелающего из .plt файла данные по набору полилиний и сохраняющего эти данные в текстовые файлы с расширением .dat

Параметры:
  • datafiles_dir –

    str

    Имя директории, в которой располагаются .plt или файлы

  • output_dir –

    str

    Имя директории для извлеченных данных

  • polylines_list –

    List[List[PolyLine]]

    список списков полилиний, по которым будут извелкаться данные; каждый список из данного списка соответствует набору полилиний для каждого файла с данными из папки datafiles_dir

  • macro_name –

    str

    имя макроса, под которым он будет сохранен

class tecplot_lib.LineDataLoader(data_dirname: str)[исходный код]¶
Базовые классы: object

Позволяет считать извлеченные по полилиниям данные из всех файлов, содержащихся в папке, путь к которой хранится в поле data_dirname, и записать данные из каждого файла в экземпляр класса Pandas.DataFrame, доступ к списку которых осуществляется через поле frames

str
